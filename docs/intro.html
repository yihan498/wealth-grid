核心理念

记账，不只是记一个数字

传统记账只告诉你花了多少、赚了多少。这里，我们回答一个更本质的问题。

01

你的财富能支撑你活多少天？

不是"账户余额多少"，而是按照你目前的生活开支，你的净储蓄可以让你自由地活多少天。这才是财富自由的真实进度。

自由天数 = 净储蓄 ÷ 日均花销
02

每笔收入都是在买时间

当你录入一笔收入，系统会立即告诉你：这笔钱买到了你多少天的自由。不是抽象的数字，而是你能"看到"的方格。

收入 → 点亮 N 个方格
03

每笔支出都是在消耗未来

支出会抬高你的日均花销，让每一元储蓄换得的自由天数变少。你会清楚地看到方格在减少——花钱的感受完全不同了。

支出 → 熄灭 M 个方格

数学模型

一个公式，把财富翻译成时间

A

日均花销基于真实行为

用你从开始记账起的所有支出除以记账天数，而不是你主观估计的"每天花多少"。越长期记账，这个数字越真实。

B

净储蓄换算自由天数

总收入减去总支出得到净储蓄，再除以日均花销，就是你"已经买到"的自由天数。方格数量幂等，永远等于这个计算结果。

C

记账天数可以自由设定

你不必从第一天就开始记账。可以设定一个开始日期，系统用这段时间内的支出数据来建立你的"历史基准线"，让模型更贴近实际。

// 核心公式
avg_daily_expense
  = total_expense / tracking_days

net_savings
  = total_income - total_expense

freedom_days
  = floor( net_savings / avg_daily_expense )

// 人生方格总数
total_cells
  = ( birth_date + target_age - today ).days

// 点亮格数
lit_count
  = min( freedom_days, total_cells )
